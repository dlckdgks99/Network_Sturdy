# 1.1 컴퓨터 통신 프로토콜의 정의 
### 통신 프로토콜의 특징
1. 효율적이고 안정적으로 통신할 수 있도록 미리 정한 약속
  - 효율적인 통신 : 주어진 통신 채널(매체, 전송 속도 등)을 최대로 이용할 수 있도록 프로토콜이 정해져야 함
    - 프로토콜의 처리 순서가 잘 정의, 패킷의 최적크기 설정, 흐름 제어
  - 안정적인 통신 : 비정상적인 외부 요인이 있어도 계속 안정되게 동작하여햐 하며 필요하다면 안전하게 통신이 종료되어야 한다.
    - 에러제어 필요

### 통신 프로토콜의 종류
1. 표준 통신 프로토콜 : TCP/IP, http, telnet, ftp, IEEE 802.3, 이더넷, 무선 LAN(IEEE 802.11)
2. 사용자 정의 통신 프로토콜 : 특정한 서비스를 제공하기 위해 개발자가 임의로 정의한 통신 프로토콜(EX 온라인 수강신청)

### OSI 7 계층 구조

2. 링크 계층

5. 세션 계층
6. 표현 계층

### TCP/IP 개요
1. TCP/IP 프로토콜 스택
2. 인터넷과 서브네트워크

### 통신 프로토콜 표현 방식
1. 필드정의
2. 파라미터 정의
3. 상태 정의
4. 동작 내용 정의
